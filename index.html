Ejemplo HTML y CSS

Contenido Principal

Este es un ejemplo básico de cómo se puede utilizar HTML y CSS juntos para crear una página web simple.

Los estilos CSS están incrustados en línea en la sección de encabezado (dentro de la etiqueta <style>), pero también podrían estar en un archivo CSS externo y vinculados desde el encabezado HTML utilizando la etiqueta <link>.
